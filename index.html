What is the DOM?

▼

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can connect to the page.

What is an event listener?

▼

An event listener is a procedure or function in a computer program that waits for an event to occur. Examples of events are user actions such as a mouse click, key press, or a change in the state of an element. The listener is programmed to react to the event by executing a callback function.
